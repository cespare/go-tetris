A simple terminal tetris implementation in Go.

## Usage:

		$ go run tetris.go

## Implemented features

* Random piece generation
* Automatic advancement
* Collision detection
* Anchoring/switching to the next piece
* Quick drop
* Line clearing
* Rotation
* Piece colors
