A simple terminal tetris implementation in Go.

## Usage:

		$ go run tetris.go

## Controls

* Move piece down: `↓`, `j`
* Move piece left: `←`, `h`
* Move piece right: `→`, `l`
* Rotate piece: `↑`, `k`
* Quick drop: `space`
* Quit: `q`, `ctrl-c`

## Implemented features

* Random piece generation
* Automatic advancement
* Collision detection
* Anchoring/switching to the next piece
* Quick drop
* Line clearing
* Rotation
* Piece colors
* Print next piece
